# xd-late-system —— 小贷贷后模块项目

当前版本：0.0.2.SNAPSHOT

# 项目框架技术

  • Web层：SpringBoot 2.0.3.RELEASE
  • Service业务层：dubbox, http
  • 数据持久层：MyBatis 3.5.2
  • Cache:Redis 2.9.0
  • DB：MySQL 5.7.21
  • DBProxy：Mycat 1.5.1
  • DataSource：Druid 1.0.15
  • MQ：RabbitMQ 5.1.2
  • RPC:Dubbo 2.7.3
  • Zookeeper 3.4.6
  • 分布式锁：Redis
  • timer：niubi-job(quartz)

# 项目架构

xd-late-system
    ├── xd-late-api         -- 对外dubbo服务接口和DTO实体定义
    ├── xd-late-common      -- 通用工具类
    ├── xd-late-data        -- 数据实体定义（不对外开放）
    ├── xd-late-service     -- dubbo接口实现和业务处理（不对外开放）
    ├── xd-late-timer       -- timer模块（不对外开放）
    ├── xd-late-web         -- http服务提供者(项目启动包)
    └── pom.xml             -- parent-pom定义模块内所有的包引用和版本定义

# 注意事项

  1. master分支只有在测试通过准备上线的时候把功能分支合并到master上
  2. 功能分支的命名方式，功能名称例如：repay
  3. 上完线需要打一个tag命名：0.0.3
  4. 大版本上完线需要把当前的版本 + 1
